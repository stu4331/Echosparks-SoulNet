## 🛠️ Technical Setup
1. Clone the repo:  
   `git clone https://github.com/stu4331/Echosparks-SoulNet.git`
2. Install dependencies:  
   `npm install`
3. Run locally:  
   `npm start`
4. Contribute via pull requests or issue discussions.
